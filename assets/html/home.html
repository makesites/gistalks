Gist#Talk


Instructions

Gist#Talk is a simple tool to help you display a Github Gist as a full blown presentation using HTML5 / CSS3.

To use any of your gists simply add the hashtag #talk in the Description field.

Remember to include at least one file that has text in Markdown.
